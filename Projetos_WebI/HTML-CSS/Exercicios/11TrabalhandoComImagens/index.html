Tipos de Imagens

Páginas para celulares:
A melhor tamanho é 300x300 e resolução 80x80.
Pode usar qualquer programa de edição de imagem para fazer isso.


Para Tablets:
tamanho 700x700 resolução 80x80.
Para notebooks:
tamanho 1000x 1000 resolução 80x80.

Inserindo Imagens Dinâmicas

Abrindo este arquivo em vários dispositivos, ou aumente e diminua o tamanho

Para isso é importante utilizar a tag picture que servirá para dimensionar as imagens de forma automática.
Neste momento há um problema, porque se for um dispositivo com tamanho de tela reduzido a imagem irá rolar, em dispositivos móveis o usuário vai embora, para corrigir isso deve usar a tag:
Source:media:type

Fazer do Max para min é mais fácil

Reprodução de Audio

Inserindo um arquivo de audio mp3, lembrando que se deve possuir licença para a reproduação dos áudios.

contudo nem todos os navegadores são compativeis com a tag audio src set então deve-se criar outra aba de audio com o parâmetro type
