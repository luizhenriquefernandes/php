Modelos de Caixas

tudo o que aparece em html são caixas uma dentro da outra e forma-se uma hierarquia de caixas
Todo o elemento visível em uma tela é uma caixa.
No CSS quano há configuração é configurado as caixas. MORFOLOGIA DA CAIXA
height - altura;
width - largura;
border para se traçar linha;
padding é um preenchimento do espaço intenor entre a borda e a fonte;
outline - é um contorno além da borda;

Tipos de caixa

box-level:
Ao terminar a linha - sempre se inicia em linha nova sempre ocupa lugar total da tela 100% do view port exemplos box level:
h1 - h6;
nav;
footer;
aside;
article;
head;
main;
p;
aside;
form;
video;

inline-level

Ao terminar a linha a linha ele monta a caixa do lado e depois para a a linha de baixo novamente.
Exemplos não quebram linha e não ocupam largura inteira da tela.
span;
a;
code;
small;
strong;
em;
sup - sub;
label;
button;
input;
select;

Diferenças

div - quando quiser quebrar a linha e criar box com a largura inteira span - quando ocupar a linha e a caixa fique ao lado
